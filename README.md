# Furniro

A Furniture Store built in top of Next.js.

## Getting Started

First, Run the `install` Command:

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
